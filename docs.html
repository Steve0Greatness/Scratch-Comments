Documentation

Welcome to Scratch Comments. An API for getting comments from the Scratch website. The only official way to currently officially get comments from some places on the Scratch website is to use them from an internal API endpoint which returns HTML, meant for making rendering easier on the developers. However, this makes it hard as a developer to do so. That is why I(Steve0Greatness) have created this API. It returns a JSON of all comments on the user's profile. Currently, the only 2 places that have an API endpoint here are users and studios.

  • Profile Comments: /users/:username
  • Studio Comments: /studios/:studio_ID

Output

The output is an Array of objects.

Keys
  • author : The poster of the comment.
  • content : HTML content of the post.
  • posted : post date, I'd recommend using new Date() on this one.
  • reply : returns a boolean for if it's a reply to the last none-reply comment.
